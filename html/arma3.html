Arma 3 Toolkit

Modpack string generator for servers commandline.

This tool generate the string needed by servers to run with mods "@mod1;@mod2".

The Mod Ignore List and checkbox states are saved in cookies, so to make it faster if your configuration happen to not be the default one.

The string will not contrain LOCALS mods, you'll need to add them manually with the folder name on your server's machine







Does your server use support spaces, - and _, check the download folder of your server to see if the mods folder contain spaces or not.
